### Accelerator Diagnostics | Beam Instrumentation | Optical Simulation | Scientific Machine Learning

I am an **M.Sc. Experimental Physics graduate** with research experience in **accelerator physics, beam diagnostics, and optical instrumentation**. * **signal extraction and image analysis for accelerator experiments**,
